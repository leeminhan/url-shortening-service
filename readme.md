### Getting Started Locally

```bash
# 1. Create venv
python3 -m venv venv

# 2. Activate venv
source venv/bin/activate

# 3. Install Dependencies
pip install -r requirements.txt
```

### Start Flask Application
```bash
python app.py
```

### Viewing the App
Go to `http://localhost:5000/` or `http://127.0.0.1:5000/`

### Testing
```bash
python -m unittest test
```
